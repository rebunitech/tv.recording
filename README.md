# TV Station Media Recorder

Pure Python script to record TV stations from screen capture

## Requirements

* Python 3.6+
* [ffmpeg](https://ffmpeg.org/)

## Usage

```bash
tv-record --help
```

## Development

```bash
git clone
cd tv.recording
pip install -r requirements.txt
pip install -e .
```
