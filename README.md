# TV Station Media Recorder

Pure Python script to record TV stations from screen capture

## Requirements

* Python 3.8+
* [ffmpeg](https://ffmpeg.org/)

## Development

```bash
git clone git@github.com:rebunitech/tv.recording.git
cd tv.recording
pip install -r requirements.txt
pip install -e .
```
## Usage

```bash
tv-record --help
```
